# Simple Portfolio

A simple, responsive personal portfolio website.

## Projects

### Project 1: Churn Modeling in Telco Business
Predicting customer churn using historical data to help improve retention strategies.
- **Demo**: `data_analysis/churn_model.py` (simulated logic)

### Project 2: Regression Analysis Predicting Stock Price
Using linear regression to forecast future stock prices based on historical trends.
- **Demo**: `data_analysis/stock_prediction.py` (simulated logic)

### Project 3: Customer Segmentation Using Clustering
Grouping customers based on purchasing behavior using K-means clustering.
- **Demo**: `data_analysis/customer_segmentation.py` (simulated logic)

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Setup
1. Clone this repository.
2. Open `docs/index.html` in your web browser.
