# Portfolio Construction Plan: Suwach Sea

## Objective
Create a modern, dark-themed portfolio website for a data analyst hosted on GitHub Pages.

## Requirements
- **Theme:** Modern dark theme.
- **Typography:** JetBrains Mono font.
- **Content:**
    - **Hero Section:** "Suwach Sea - Data Analyst".
    - **Projects Section:** Display 3 data analysis projects.
    - **Contact Section:** Display contact details.
- **Hosting:** GitHub Pages.

## Implementation Steps

### 1. Project Setup
- [ ] Initialize a new directory structure:
    - `/` (root)
        - `index.html`
        - `style.css`
        - `script.js`
        - `/assets` (for images or project thumbnails)

### 2. Base Design & Typography
- [ ] Implement `index.html` with basic structure.
- [ ] Add Google Fonts (JetBrains Mono).
- [ ] Set up `style.css` with a modern dark color palette.

### 3. Content Development
- [ ] Develop the **Hero Section** with Name and Title.
- [ ] Create the **Projects Section** with 3 project card components.
- [ ] Add the **Contact Section** with social/contact links.

### 4. Deployment
- [ ] Create a GitHub repository.
- [ ] Commit and push local files to `main` branch.
- [ ] Configure repository Settings -> Pages for deployment.

## Verification
- [ ] Verify site responsiveness.
- [ ] Confirm dark theme and font are applied correctly.
- [ ] Test links in the Contact section.
- [ ] Access the site via the published GitHub Pages URL.
